# YouDL
Application for downloading audio, video, and subtitles from YouTube. easily input a YouTube URL, select the desired format (MP3, MP4, or Subtitles), and start the download.

# Features
- Audio Download
- Video Download
- Subtitles Download
- Format Selection
- clean GUI built with Tkinter
